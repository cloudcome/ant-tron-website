身份识别专家
为各行业提供基于指纹识别技术的应用解决方案
  • 指纹终端
    包含指纹采集终端、指纹采集模块、指纹验证终端。在物理结构上兼容多种指纹传感器，如电容式FPC1011、FPC1020、TCS、光学等。具备自主知识产权的活体检测算法、能够自动处理不同干湿手指；支持在线和离线等多种使用场景。
  • 指纹应用
    是为各行业提供、安全、快捷的现代化工作和生活方式的指纹应用场景管理的综合性平台。
  • 指纹云
    在多年指纹识别技术积累的基础上，依托蚁创私有云技术，可为大部分细分行业提供数据分析、数据终端接口。
  • 解决方案
    提供Android平台指纹应用解决方案，支持多种指纹图像，如160*160,128*128*，128*80，96*96。匹配时间小于1秒，FAR<0.01%,FRR<1%。
无线温湿度验证系统
全球领先的无线温湿度验证系统提供商，广泛用于制药、食品、医疗、运输等行业
  • 数据采集器
    是一个非常小的无线独立部件，可以自动按设定的时间间隔探测关键的参数，如温度，湿度，然后记录在安全芯片中，并由自带电池持续供电。有提供温度、压力和温度三种数据采集器。
  • 数据交换盒
    可以将数据采集器放入其中，通过专用数据线与电脑相连接，将数据从采集器中传输到电脑，另外也可用于完成数据采集器的初始化设置。
  • 验证系统
    基于 Windows 平台的软件，能够自适应多款数据采集器，完成数据收集、分类、分析、报告等强大功能。
  • 检测中心
    正向国家相关单位申请检测单位许可，敬请期待！
